# 🌊 AquaSense – Water Quality Prediction System

## 🚀 Overview

**AquaSense** is an AI-powered water quality prediction platform that leverages **IoT sensor data** and **machine learning models** to analyze and predict water quality in real time.

The system helps monitor critical parameters like **pH, temperature, and turbidity**, enabling smarter environmental decisions and safer water management.

---

## 🧠 Key Features

* 📡 **Real-time Data Processing**
  Collects water data via IoT sensors (pH, temperature, turbidity)

* 🤖 **AI-Based Prediction**
  Uses machine learning models to predict water quality metrics such as dissolved oxygen

* 📊 **Interactive Dashboard**
  Clean UI to visualize inputs and prediction results

* ⚡ **Fast & Scalable Deployment**
  Hosted on Vercel for high performance and reliability

* 🌍 **Environmental Impact**
  Helps in smart water monitoring for agriculture, aquaculture, and urban systems

## 🧩 System Architecture

1. IoT Sensors collect real-time water data
2. Data is preprocessed and cleaned
3. Machine learning model predicts water quality
4. Results are displayed on the web dashboard
